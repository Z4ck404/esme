Data storage, sharing & security

A course to introduce you to the field of cyber security in general and in the context of connected cities and IoT projects in particular. This course is basically for beginners and does not require a lot of advanced computer knowledge, but some knowledge is needed in IoT to know how to manipulate the arduino in addition to some basic knowledge in python.

Course Objectives

  1. Data collection and processing (IoT, mobile and web apps, etc.)
  2. Cyberattacks risks and challenges (Data breaches, etc).
  3. IoT & smart city monitoring (extending your existing projects)
  4. Data protection & privacy
  5. Cybersecurity roles and career paths

Practical session

The code to connect to Arduino using python

Before runing the code, please read the following :
  1. Make sure to have python installed in your machine :
    2. To know if python is install, open a command line and type python ! if you get an error such as python is not known, then install python as explained in this tutorial.
  2. Install the necessary python packages ( pip) and liberaries
    3. - Install pyserial with this command in your terminal : pip install pyserial
    - Install elasticsearch python liberary : pip install elasticsearch
  3. Install and run elasticsearch in your machine
    1. Download the .zip
    2. Go to /bin and run elasticsearch.bat. You should see a terminal window showing some logs, leave it open
    3. Go to your browser and open http://localhost:9200, if it showed something like :
